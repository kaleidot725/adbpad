AdbPad

A GUI tool to streamline Android app testing using ADB

Available for Windows & macOS

✨ Features

Device Management

View a list of connected Android devices and switch between them effortlessly

ADB Command Execution

Run ADB shell commands with ease

Text Input

Send text input to your Android device

Screenshots

Capture screenshots for each theme

Virtual Display

Create virtual displays to test on large-screen environments

⬇️ Installation

Via Homebrew (macOS)

brew tap kaleidot725/kaleidot725
brew install --cask adbpad

Manual Installation

  1. Download the latest installer from the Releases page
  2. Launch the application and configure the ADB path in Settings
For macOS users: Since the app is not signed with an Apple Developer account, you may encounter a security warning stating that the developer cannot be verified. To open the app:
  1. Right-click (or Control-click) the AdbPad icon and select Open
  2. In the dialog that appears, click Open again
  3. Alternatively, go to System Settings > Privacy & Security and click Open Anyway
