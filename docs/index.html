AdbPad

ADBを使ったAndroidアプリテストを効率化するGUIツール

Windows & macOS対応

✨ 機能

デバイス管理

接続されているAndroidデバイスの一覧を表示し、簡単に切り替えができます

ADBコマンド実行

ADBシェルコマンドを手軽に実行できます

テキスト入力

Androidデバイスへのテキスト入力を送信できます

スクリーンショット

各テーマのスクリーンショットをキャプチャできます

バーチャルディスプレイ

大画面環境でのテストに便利なバーチャルディスプレイを作成できます

⬇️ インストール方法

Homebrew経由でのインストール (macOS)

brew tap kaleidot725/kaleidot725
brew install --cask adbpad

手動インストール

  1. Releasesページから最新のインストーラーをダウンロード
  2. アプリケーションを起動し、設定でADBパスを設定
macOSユーザーへの注意: このアプリはApple Developer アカウントで署名されていないため、セキュリティ警告が表示される場合があります。アプリを開くには:
  1. AdbPadアイコンを右クリック（またはControlキーを押しながらクリック）し、「開く」を選択
  2. 表示されるダイアログで再度「開く」をクリック
  3. または、「システム設定 > プライバシーとセキュリティ」から「このまま開く」をクリック
