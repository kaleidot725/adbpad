Features

Everything you need for Android testing

All the essential ADB workflows, wrapped in a clean desktop UI.

📱

Device Management

View all connected Android devices at a glance and switch between them with a single click.

⚡

ADB Command Execution

Run ADB shell commands directly from the UI without opening a terminal window.

⌨️

Text Input

Send text input to your Android device instantly — great for filling forms during testing.

📸

Screenshots

Capture device screenshots for each theme with a single click and save them locally.

🖥️

Virtual Display

Create virtual displays via scrcpy to test your app on large-screen environments.

Installation

Get started in minutes

Choose the installation method for your platform.

🍎

macOS — Homebrew

brew tap kaleidot725/kaleidot725
brew install --cask adbpad
⚠️
Gatekeeper warning: The app is not signed with an Apple Developer certificate. Right-click the icon → Open, then click Open in the dialog. Or go to System Settings › Privacy & Security and click Open Anyway.
🪟

Windows / Manual

  1. Download the latest installer from the Releases page.
  2. Run the installer and follow the on-screen instructions.
  3. Launch AdbPad and configure the ADB path in Settings.
